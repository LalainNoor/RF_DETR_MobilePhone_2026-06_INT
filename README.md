# RF-DETR Mobile Phone Detection

Dataset: Custom Mobile Phone Dataset (COCO format)

Model: RF-DETR

Framework: PyTorch + PyTorch Lightning

Dataset Split:
- Train: 56
- Validation: 16
- Test: 8

Hardware:
- Google Colab
- NVIDIA Tesla T4 GPU

Task:
Train and evaluate RF-DETR for mobile phone object detection.
